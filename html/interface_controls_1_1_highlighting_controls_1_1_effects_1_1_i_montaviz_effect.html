Detailed Description

Interface for MontavizEffects, Effects inheriting from this Interface can be used in HierarchicalEffectManager. An Effect is not linked to a GameObject but to a set of renderers. Normally only on Effect-Instance is created per Model. The renderers are wrapped in WrappedRenderer objects, therefore it is possible to link data to them.
